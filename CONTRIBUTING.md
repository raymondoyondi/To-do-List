I welcome contributions to make this calculator even better! If you have suggestions or find any bugs, please follow these steps:

1. Fork the repository (https://github.com/raymondoyondi/Task-Manager/fork).

2. Create your feature branch (git checkout -b feature/AmazingFeature).

3. Commit your changes (git commit -m 'Add a fantastic new feature').

4. Push to the branch (git push origin feature/AmazingFeature).

5. Open a Pull Request detailing your changes.
